# Administrator guide

Sign in at <http://localhost:3001/admin/sign-in>. Correction reviewers use
`/admin/corrections`; verifiers use `/admin/reverification`. Never share administrator accounts.
Actions require the current record version and are audited. Correction content is untrusted text;
do not copy it into public records without independent evidence. Reporter contact is available only
to administrators with explicit contact permission and must not be exported casually.
