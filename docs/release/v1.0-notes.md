# CivicSignal v1.0 release notes — draft

CivicSignal is a governed community-resource publishing platform with public search and correction
reporting, secure role-based administration, re-verification tasks, immutable revisions, source
provenance, evidence-gated publication, freshness tracking, stale detection, and audit history.

The current release candidate includes responsive public and administrator interfaces, automated
accessibility checks, opaque sessions, CSRF protection, Argon2 password hashing, optimistic
concurrency, Docker development, PostgreSQL migrations, and Render deployment configuration.

The hosted mode must remain a clearly labeled fictional portfolio demonstration until a small
real-data pilot has completed human source review. The optional Washington, DC candidates cover
five services from official District sources and expire for review after 14 days.

Known limitations: incomplete geographic coverage, no capacity guarantees, basic text search, no
geocoded distance ordering, process-local public-write rate limiting, and no completed production
operations program. This draft is not a claim that CivicSignal is a live public-service directory.
