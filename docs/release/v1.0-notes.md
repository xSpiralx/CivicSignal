# CivicSignal v1.0 release notes — draft

CivicSignal is a governed community-resource publishing platform with public search and correction
reporting, secure role-based administration, re-verification tasks, immutable revisions, source
provenance, evidence-gated publication, freshness tracking, stale detection, and audit history.

The current release candidate includes responsive public and administrator interfaces, automated
accessibility checks, opaque sessions, CSRF protection, Argon2 password hashing, optimistic
concurrency, Docker development, PostgreSQL migrations, an audited source registry, bounded
CSV/JSON imports, deterministic deduplication, SSRF-safe outbound retrieval, public trust pages, and
free-tier Vercel/Render/Neon deployment configuration.

The geographic model and interface support all 50 states, the District of Columbia, and populated
U.S. territories, including city/town, postal-code, and service-area filtering. The hosted mode must
remain no-indexed and clearly labeled as a fictional portfolio demonstration until an initial set of
real records has completed named human source and record review. Eight Massachusetts official-page
candidates form one regional research queue; zero are approved, imported, verified, or published.

Known limitations: incomplete geographic coverage, no capacity guarantees, basic text search, no
geocoded distance ordering, process-local public-write rate limiting, and no completed production
operations program, free-tier cold starts, and no current public deployment. This draft is not a
claim that CivicSignal is a live public-service directory.
