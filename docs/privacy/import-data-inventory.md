# Import data inventory

Candidates contain public organizational and program facts: names, public phone/email/website, service descriptions, addresses, cities, counties, states, ZIP codes, service areas, hours, categories, eligibility, costs, languages, accessibility, application instructions, source identifiers, source dates, licenses, hashes, and review metadata.

The HRSA wave contains facility records, not patients, cases, or individual demographics. Unpublished candidates are administrator-only and excluded from public APIs, search, sitemaps, and statistics. Raw projections are retained only where source terms allow and must not contain sensitive narratives.
