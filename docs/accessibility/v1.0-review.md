# v1.0 accessibility review

Automated axe checks found no serious or critical WCAG A/AA issues on the landing page, directory,
or administrator sign-in in desktop Chromium and a Pixel 5-sized mobile viewport. Public search,
details, correction reporting, dialog focus containment/restoration, semantic landmarks, labels,
status regions, and touch-sized controls have automated coverage.

Stable v1.0 still requires a complete manual keyboard pass across the administrator workflow,
VoiceOver spot checks, 200% zoom, reduced-motion verification, long-content checks, and representative
320, 375, 390, and tablet widths. CivicSignal does not claim WCAG certification.
