# Source license review template

- Stable identifier, publisher, dataset and jurisdiction
- Landing, download, license and terms URLs
- Redistribution, modification and automated-access permissions
- Required attribution and retention limits
- Authentication, rate limits and hostname allowlist
- Latest source update, refresh cycle and record-level date semantics
- Categories, geography, known omissions and sensitive fields
- Legal reviewer, technical reviewer, review dates and decision
- Decision: proposed, license review, technical review, approved, restricted, rejected, or disabled

Engineering review is not formal legal advice. Unclear permission results in restriction, not ingestion.
