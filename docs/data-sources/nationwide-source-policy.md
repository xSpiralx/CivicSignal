# Nationwide source policy

CivicSignal ingests only government open data, explicitly licensed nonprofit data, authorized APIs, provider-authorized submissions, and datasets whose terms expressly permit reuse. Public visibility is not permission to copy a directory.

An enabled source requires a named administrator decision, documented redistribution and modification rights, an allowlisted host, a freshness policy, and legal and technical review dates. Google Maps, Yelp, Facebook, Findhelp, unlicensed 211 data, search results, restricted pages, CAPTCHA-protected sources, and sources with unclear reuse rights are blocked.

Imports are private candidates. No source decision, import, duplicate classification, or bulk action publishes a resource.
