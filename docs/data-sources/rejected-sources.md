# Rejected and restricted sources

| Source | Decision | Reason |
| --- | --- | --- |
| Findhelp | Rejected | Proprietary directory; no redistribution permission supplied |
| Google Maps, Yelp, Facebook | Rejected | Consumer-platform data and contributor content are not an authorized CivicSignal source |
| 211 directories | Restricted | Requires valid API or written redistribution authorization |
| Mass.gov program pages | Restricted | Manual factual review only until page-specific reuse rights are documented |
| Municipal service pages | Restricted | Public access and robots permission do not establish a reusable dataset license |
